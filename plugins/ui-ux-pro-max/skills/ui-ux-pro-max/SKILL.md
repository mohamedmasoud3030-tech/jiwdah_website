# UI/UX Pro Max

This local wrapper installs the `ui-ux-pro-max` plugin from `nextlevelbuilder/ui-ux-pro-max-skill` for Codex plugin discovery in this repository.

Use it when a task changes how an interface looks, feels, moves, or is interacted with, including new pages, landing pages, dashboards, admin panels, e-commerce flows, SaaS screens, portfolios, mobile UI, and reusable components.

## Working Rules

- Start with the product context before choosing style, color, type, spacing, motion, and interaction patterns.
- Prioritize accessibility, touch targets, readable typography, keyboard navigation, focus states, and contrast before visual flourish.
- Use an intentional visual direction rather than generic AI-looking gradients or interchangeable SaaS layouts.
- Validate responsive behavior at mobile, tablet, laptop, and desktop widths.
- Use SVG/icon libraries for icons instead of emoji when implementing production UI.
- Keep animations meaningful, short, interruptible, and compatible with `prefers-reduced-motion`.
- For dashboards and data-heavy UI, make legends, tooltips, labels, and non-color indicators clear.
